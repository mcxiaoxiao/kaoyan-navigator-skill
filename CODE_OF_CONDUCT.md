# 社区行为准则

## 我们的承诺

我们致力于为所有参与者提供友善、包容、无骚扰的协作环境，不因年龄、身体状况、民族、性别认同、经验水平、教育背景、地域或其他身份特征区别对待。

## 倡导的行为

- 尊重不同的择校目标、教育背景和风险偏好；
- 对数据、方法和代码提出具体、可验证的意见；
- 接受合理的纠正，并及时修复错误；
- 保护考生隐私，不传播未经授权的个人资料；
- 将讨论聚焦于项目改进。

## 不可接受的行为

- 人身攻击、歧视、骚扰或威胁；
- 嘲讽学校层次、本科背景、考试成绩或个人选择；
- 发布他人的准考证号、联系方式等隐私信息；
- 故意提交虚假招生数据、来源或预测结论；
- 垃圾推广、引流和与项目无关的商业宣传。

## 执行

维护者可删除不当内容，并对持续违反准则的参与者采取警告、临时限制或永久禁止参与等措施。

报告行为问题时，请通过仓库维护者公开资料中提供的私密联系方式联系。报告内容将尽量保密处理。

本准则参考 Contributor Covenant 的基本原则，并针对考研数据和隐私场景进行了调整。
